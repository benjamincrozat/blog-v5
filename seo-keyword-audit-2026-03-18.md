## How to read this

- `GSC` uses `clicks / impressions / avg position` from the fresh export.
- `No exact row` means the keyword did not appear as an exact query in the top 1,000 exported GSC rows.
- `Optimize now` means the page-query fit is strong enough to justify a real refresh.
- `Secondary only` means we should support the keyword inside the current page, but not force it to become the primary target.
- `Deprioritize` means the query is low-value, mismatched to the current page, too navigational, or too dominated by official sources to be a good use of effort right now.
